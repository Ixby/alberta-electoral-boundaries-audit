## 3. CRS notes

- 2019 EDs use EPSG:3401 but 2023 VAs use EPSG:3400. Both are NAD83 / Alberta TM projections with identical datum; reprojection is trivial. For spatial joins in later phases, reproject both to a common CRS (recommend EPSG:3400 since it covers the entire province without zone boundaries issues).
- 2021 Census DAs (see `alberta_2021_das.gpkg`) use EPSG:3347 (Statistics Canada Lambert). Reproject to match the electoral layers before spatial joins.
